Score Formula

คะแนนนี้เป็น proxy score เพื่อเรียง strength signal และ improvement priority ไม่ใช่อัตราผ่าน

achievement_score = 100 x weighted_signal
  • 20%passed volume
  • 20%school spread
  • 20%average total score
  • 15%average A3
  • 10%high-score share
  • 7.5%low borderline share
  • 7.5%low A3-zero share

แต่ละ component ถูกแปลงเป็น percentile signal ในกลุ่มจังหวัดก่อนนำมาถ่วงน้ำหนัก: ค่าใกล้ 1 หมายถึงสัญญาณดีกว่าจังหวัดส่วนใหญ่ ค่าใกล้ 0 หมายถึงควร review มากกว่า

rank_high / rank_low rank_high เรียง achievement_score จากสูงไปต่ำ ส่วน rank_low เรียงจากต่ำไปสูงเพื่อหา improvement priority
percentile signal ใช้ลำดับ percentile ของจังหวัด ไม่ใช่ค่าดิบโดยตรง เพื่อให้ metric คนละ scale รวมกันได้ในสูตรเดียว
inverse percentile ใช้กับ metric ที่ยิ่งต่ำยิ่งดี เช่น total_40_share_pct และ a3_zero_share_pct ดังนั้น share ต่ำจะได้ signal สูง

Metric Glossary

ความหมายของตัวเลขหลักในตารางและกราฟ

passed_students จำนวนผู้ผ่านเกณฑ์ในจังหวัดนั้นจาก dataset นี้ volume_signal = percentile(passed_students)
normalized_schools จำนวน school_norm_key ที่พบในจังหวัด หลัง normalize ชื่อโรงเรียนแบบ conservative school_spread_signal = percentile(normalized_schools)
avg_total_all ค่าเฉลี่ยคะแนนรวมของผู้ผ่านในจังหวัดนั้น avg_score_signal = percentile(avg_total_all)
avg_a3 ค่าเฉลี่ยคะแนนหมวด A3 ซึ่งเป็น bottleneck สำคัญใน brief a3_signal = percentile(avg_a3)
high_score_60_plus_share_pct สัดส่วนผู้ผ่านที่มี total_all ตั้งแต่ 60 คะแนนขึ้นไป high_score_signal = percentile(high_score_60_plus_share_pct)
total_40_share_pct สัดส่วนผู้ผ่านที่ได้ total_all = 40 ซึ่งเป็นกลุ่ม borderline ผ่านพอดี low_borderline_signal = 1 - percentile(total_40_share_pct)
a2_a3_20_share_pct สัดส่วนผู้ผ่านที่มี A2 + A3 = 20 ใช้เป็น bottleneck context แต่ไม่อยู่ในสูตร achievement_score
a3_zero_share_pct สัดส่วนผู้ผ่านที่ได้ A3 = 0 ยิ่งสูงยิ่งสะท้อนคอขวด A3 low_a3_zero_signal = 1 - percentile(a3_zero_share_pct)
